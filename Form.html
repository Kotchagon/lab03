    การใช้งาน Grid ใน Bootstrap

    Post by Zomzom, Feb 8, 2017.

    WORK

    Grid System คืออะไร ?

    การใช้งาน Bootstrap นั้น ต้องเริ่มต้นจากการสร้าง Containerและตามด้วยจัดแบ่งสัดส่วน Column ด้วย หลักการ Grid System



    ซึ่งหน่วยที่เป็น 12 นี้ เราสามารถเฉลี่ยแต่ละส่วนได้ตามความต้องการ อย่างเช่นGrid System ของ Bootstrap ปกติแล้ว grid system ที่ Bootstrap ให้มาจะเป็ นแบบ 12 คอลัมน์ ซึ่งจะใช้ร่วมกับ container ที่เพิ่ ง สร้างขึ้นมา โดยแต่ละคอลัมน์จะมีความกว้างเท่ากัน 60px และจะอยู่ห่างกัน 20px จะได้ว่า (60*12) + (20*11) = 940) หรือความกว้างของ containerการจะสร้าง layouts ด้วย Bootstrap ให้ สร้าง html element ที่มี class “row” ขึ้นมาก่อนถึงจะสามารถสร้างแต่ละคอลัมน์ขึ้นมาได้ด้วยการ ใส่html element ที่มี class “span*” เข้าไปข้างใน โดย * ที่อยูข้างหลัง span นั้นจะเป็นตัวเลขที่จะ ระบุว่า ต้องการให้คอลัมน์นี้ขนาดเท่าไร สมมติว่า จะสร้าง layouts แบบ 12 คอลัมน์ ให้ใส่โค้ด ดังนี้



    จากโค้ดด้านบนได้ใส่ HTML elements ที่มี class ชื่อ “span1”รวมทั้งหมด 12 elements ด้วยกัน เมื่อพรีวิวดู container จะถูกแบ่งออกเป็นคอลัมน์ต่างๆ ที่มีความกว้างเท่ากันทั้งหมดจํานวน 12คอลัมน์ ด้วยกัน 2 คอลัมน์ ถ้าต้องการแบ่ง 2 คอลัมน์ และให้คอลัมน์ซ้ายใหญ่กว่าคอลัมน์ขวาดังนี้



    จะเห็นวาความกว้างของทั้ง 2 คอลัมน์นี้จะเป็น 8+4 = 12 พอดี ที่จําเป็นต้องกาหนดความกว้างให้ รวมกันเท่ากับ 12 เพราะว่า grid system ของ bootstrap นั้นเป็นแบบ 12 แล้วเท่ากับ 2 คอลัมน์นี้จะต้องเต็ม container พอดี 3 คอลัมน์ ถ้าต้องการ 3 คอลัมน์ ก็กำหนดให้ 3 คอลัมน์มีผลรวมเท่ากับ 12 ตามเดิม


    จะเห็นว่าความกว้างของทั้ง 2 คอลัมน์นี้จะเป็น 8+4 = 12 พอดี การที่เราจําเป็นต้องกำหนดความกว้างให้รวมกันเท่ากับ 12 เพราะว่า grid system ของ bootstrap นั้นเป็นแบบ 12 แล้วเท่ากับว่า 2 คอลัมน์นี้จะเต็ม container พอดี Offsetting columns ในบางครั้ง ถ้าต้องการเพิ่มระยะห่างระหวางคอลัมน์ เราสามารถทําได้ด้วยการเพิ่ม class ที่ชื่อ “offset*” เข้าไป โดยจะหมายถึงจํานวนของคอลัมน์ที่จะกลายมาเป็ นระยะห่างดังนี้


    เมื่อพรีวิวจะพบวาทั้้ง 2 คอลัมน์นี้ยังคงเต็ม container อย่เนื่องจาก 2 + 4 + 6 = 12 Nesting columns


    นอกจากนั้นยังสามารถสร้างคอลัมน์ซ้อนภายในคอลัมน์ได้ด้วยตามตัวอย่างนี้จะเห็นว่าได้สร้าง คอลัมน์“span2″ กับ “span6″ ซ้อนข้างในคอลัมน์ “span8″ ซึ่งคอลัมน์ที่อยูภายในทั้ง 2คอลัมน์จะเต็มคอลัมน์ข้างนอกพอดีเนื่องจาก2 + 6 = 8
